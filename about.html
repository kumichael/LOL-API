About the LoL API

The LoL API project was started on June 25th 2012 as a summer project. The purpose of this was to create a fully functioning API in which users could access all the information about their favorite League of Legends champions and items for use as content on a fan site.

This project was started because of the lack of an official API when I first created a mobile LoL guide site for personal use. I looked around the web and could not find an API dealing with League of Legends. So I decided to create one.

Just as a disclaimer, please don't expect too much from this project. I do plan on refining it and adding functionality to it once it is finished; however, this is a one-person project started by and maintained by myself alone.

I generally used the League of Legends Official Website for champion statistics such as Health rating, Attack rating, etc. However, it seems some of the skill data on the official site is outdated. Because of this, I use both Mobafire and the League of Legends Wikia to double-check skill information.
