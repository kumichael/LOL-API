GET champions

Return information about the champion selected.

Resource URL
http://lol.michaelku.com/api/champion/
ResourceDescription
name
required
Tells the API which champion's information to retrieve.
skills
optional
When set to 'true', the API will return the skill list of the champion. Linebreaks are denoted by asterisks (*) for easy string splitting.
This is set to 'false' by default.
format
optional
Tells the API what format to return the data in--either json or xml.
This is set to 'json' by default.
Example Requests
GET champion 'Teemo' in default JSON format
http://lol.michaelku.com/api/champion/?name=teemo 
{
    "champion":{
        "id":"17",
        "name":"Teemo","
        title":"The Swift Scout",
        "attributes":"Ranged, Stealth, Support",
        "attack":"6",
        "health":"4",
        "magic":"8",
        "difficulty":"5",
        "ip":"1350",
        "rp":"585",
        "release":"2009-02-21"
    }
}
GET champion 'Teemo' in XML format
http://lol.michaelku.com/api/champion/?name=teemo&format=xml

    <id>17</id>
    <name>Teemo</name>
